Edge

Means local (or near local) processing.

Cloud

Data Center, literally.

Edge vs Cloud processing

When using cloud, there needs to be back and forth of http handshake to process the data,
but edge device can process whatever it needs on spot.

For example, AI algorithm can be processed in the cloud, and get sent to the edge device for action.

But the thing is, edge device isn't defined by its characteristics, but its benefits.

For example, self-driving cars, that has to be an edge device or things could get kinda dangerous;
I mean, if it's cloud running, in a middle of a high way, the disaster could be news worthy.

But gaining insights via machine learning algorithm from the millions of sales transactions would have no problem of being cloud run.
